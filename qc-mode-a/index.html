Batasan Mode A

Mode A hanya mengecek data struktur & teks dari Google Forms API: kelengkapan field, status wajib-diisi, urutan section, dan tipe pertanyaan. Google Forms API tidak menyediakan data bold/italic/warna/tema/header visual, sehingga hal-hal tersebut tidak dicek di sini. Untuk cek visual, gunakan Mode B (upload screenshot + OCR) secara terpisah.
