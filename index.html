QC Otomatis Survei Google Form

Non-AI · Rule-Based · Static Site

Pilih mode pemeriksaan sesuai kebutuhan Anda. Kedua mode berjalan sepenuhnya di browser (tidak ada server/database), dan sama-sama murni rule-based tanpa AI/LLM.

🔗

Mode A — Cek via Link Google Form

Login dengan Google, paste link editor form Anda. Data diambil langsung dari Google Forms API — cek kelengkapan field, status wajib-diisi, urutan section, dan tipe pertanyaan. Akurat 100% untuk hal-hal struktural.

  • ✅ Butuh login (hanya untuk pemilik/kolaborator form)
  • ✅ Tidak bisa cek bold/italic/tampilan visual
  • ✅ Hasil: tabel temuan + PDF ringkas
Buka Mode A →
🖼️

Mode B — Cek via Upload Screenshot/PDF

Upload screenshot atau PDF tampilan form (dari siapa pun, tanpa perlu akses Editor). Sistem membaca teks dengan OCR dan mendeteksi format visual (bold/italic) dengan pengolahan citra klasik — bukan AI.

  • ✅ Tidak perlu login / akses ke form asli
  • ✅ Bisa cek bold/italic (heuristik, best-effort)
  • ✅ Hasil: PDF beranotasi kotak merah + catatan
Buka Mode B →
Belum yakin pilih yang mana? Kalau Anda pemilik form dan cuma butuh cek cepat kelengkapan field, pakai Mode A. Kalau Anda mengecek form milik orang lain (tanpa akses login) atau butuh cek tampilan bold/italic, pakai Mode B.
